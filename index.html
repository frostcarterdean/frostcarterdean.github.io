COVID-19 Travel Guide

Website that links to breaking news articles about COVID-19, displays COVID-19 statistics for all states and counties in the United States, and shows statistics and information about COVID-19 worldwide. Made with vanilla HTML, CSS, JavaScript, PHP, and MySQL and uses data from four different APIs.

Home page uses PHP, mySQL, and manual user entry to alter what content is displayed; the site owner uses a form to change information outputted in one of the seven templates.

United States page pulls data from COVID Tracking Project Data API and Covid Act Now API for statewide and county information respectively. It contains an interactive map courtesy of Google GeoCharts; the user can hover over states to glance data and click on them to view county data.

The World page pulls data from Oxford COVID-19 Government Response Tracker API and Coronavirus COVID19 API. Data from the former is used to output a sortable and filterable table with data for various countries. The latter is used to display overall global statistics.

Group project made as part of studies. Home page, all of United States page except some map implementation, and all of World page are my personal work; logo and part of map implementation are courtesy of other students.
